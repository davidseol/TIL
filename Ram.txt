Ram(randon access memory)
:
임의로 접근하는 메모리(?)

우리가 쓰는 코드는 컴퓨터의 메모리(ram)에 올라가게된다.

우리가 쓰는 java코드(소스코드)가 컴파일되어 메모리코드로 컴퓨터에 전달되는데(메모리) 메모리의 영역은 3군데로 나눌수 있다 

메서드영역
-기본 내장 java 메서드
-import 패키지/클래스
-작성한 클래스 class

스택영역
-코드 블럭 단위로 생성, 중괄호 기준으로 시작과 끝 판단
-지역변수,매개변수(인자,args)

힙 영역
-프로그램에 사용 되는 객체


메모리가 각각의 영역들이 있으며 과하게 생성됐을때는 터지게 되어 프로그램이 돌아가지 않는 상황이 발생할수있다.

클래스가 올라가는 메서드영역과 메서드가 올라가는 스택영역은 용어의 혼용이 없도록 구분해서 기억하자
